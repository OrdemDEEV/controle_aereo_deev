# Controlador de Tráfego aéreo

Projeto com instuito de simular um controlador de tráfego aéreo,
para possivelmente melhorar o controle de software na área da aviação civil.

 <h4>Tecnologias Utilizadas</h4>
 
  - Java / JSF.
  - Hibernate.
  - Bootstrap.
  - Html.
  - Canvas.
